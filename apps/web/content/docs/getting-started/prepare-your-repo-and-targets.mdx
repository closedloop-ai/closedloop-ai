---
title: Prepare Your Repo and Targets
description: Make local execution predictable before you scale up docs or code generation.
---

## Repository checklist

- default branch is stable
- contributor permissions are clear
- there is a clean branch strategy for loop output
- core app commands are documented

## Compute target checklist

- the desktop client is installed and healthy
- required CLIs are available
- the target reports provider support accurately
- network access is sufficient for the task

## Artifact checklist

- the artifact has a clear title
- the goal is concrete
- success criteria are reviewable
- context is attached where needed

When these three layers are ready, loops are much less likely to fail for environmental reasons.
