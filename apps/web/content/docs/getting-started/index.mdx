ClosedLoop.ai works best when the initial setup is treated as system configuration, not as a one-time demo.

## What you need

1. A repository with clear ownership and a meaningful first task.
2. The API and app running so you can create or review artifacts.
3. At least one compute target ready to execute loops.
4. A habit of treating docs, PRDs, plans, and features as linked artifacts.
