---
title: Artifact
description: A durable object that defines or records work in ClosedLoop.
---

An **artifact** is any durable object the platform recognizes: a PRD, an implementation plan, a ticket, a judge report, a loop run, a context pack, or an uploaded template.

Artifacts are versioned, addressable, and reviewable. Execution attaches to them. Review closes against them. They are the system of record.

See [Artifacts](../mechanisms/artifacts) for the full map of on-disk and platform artifact types.
